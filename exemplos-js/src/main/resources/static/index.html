Apresentar na tela os dados pessoais da dupla carregados a partir do arquivo JSON

Criar o arquivo dados.json na mesma pasta deste arquivo index estruturado com as seguintes informações dos dois alunos:

  • Nome completo
  • Foto (nome do arquivo)
  • E-mail
  • Telefone
  • Data nascimento
  • Link do Github
  • Link do Linkedin

A foto pode ser copiada manualmente no diretório /img já existente

Escrever o código Javascript para carregar os dados do JSON usando Fetch API e apresentar dentro da tela definida neste arquivo

Os dados podem ser carregados juntamente com a tela no navegador ou então através de algum evento (ex: clique em botão)
